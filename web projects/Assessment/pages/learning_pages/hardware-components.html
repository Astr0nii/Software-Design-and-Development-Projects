Hardware Components

The hardware components of a computer are the most fundamental elements that make up your computing device. It is the physical equipment involved in processing information. Hardware is given five main functions to perform: input, output, processing, storage and control. Input
Input involves entering data into a computing device for processing, this data could be a number, button press, name or mouse movement. If the data entered doesn't match what the hardware expected then you'll be presented with a meaningless output, also known as 'garbage in, garbage out' (GIGO). Some input devices are: keyboards, drawing tablets, console controllers, mice, microphones, scanners, joysticks and disks. Output
Ouput involves presenting processed data to a person, or maybe to another computer. this functions also includes the transfer of data from primary storage to the appropiate output device, this may be a printer or speaker. Process
Process is how the data is changed to produce information, this change is commanded by a series of instructions. Processing is carried out by the central processing unit (CPU). The CPU takes data from an input device, performs some set of instructions, and then sends the new data to an ouput device. Storage
Storage involves receiving and retaining data over a set period of time. It can be accessed whenever required and it is catergorised into two types: primary and secondary. Primary storage holds data and programs before and after they are processed by the CPU. Secondary storage is purpose built for longer-term storage solutions, it can store large amounts of data permanently and is usually called 'external storage'. It uses media such as: Hard disks, Solid State Disks and DVDs to store data. Control
Control is the glue that holds everything together, it coordinates the flow of data and the other operations listed above.

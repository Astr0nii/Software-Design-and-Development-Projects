System and Application Software

Software can be catergorised into to groups, system software or application software. These can be divided into the more groups which will be covered later on.
System Software
System software can be further divided into these two groups: operating system and utility software. Utility software allows a user to delete files, or create folders, manage shortcuts, antivirus protection and license installation.

Theoperating system software, also known as a driver is responsible for how your hardware communicates with the operating system, it tells your graphics card or any other hardware componenthow to send the correct data for processing and where to send it. Without drivers your hardware would be useless!
Application Software
Appplication software can also be further divided into two groups: off-the-shelf software and custom software. You can think of off-the-shelf software as: microsoft word, chrome, your favourite video game, netflix app and more, this group of software is readily avaliable to the public, which means there should be considerations for a large varied userbase.

Custom software is a one-off solution, it is purpose built for a specific reason. For example, Coles has custom software for their checkout kiosks, this software won't be found anywhere else and was purpose built for coles to use.
